Detailed Description

L'interfaccia CheckerEmail definisce un contratto per le classi che devono implementare la logica di validazione di un indirizzo email. Ogni classe che implementa questa interfaccia dovrà fornire una propria implementazione del metodo isValidEmail(), che permette di verificare se un indirizzo email è valido.

L'interfaccia è destinata ad essere implementata da classi che rappresentano entità che hanno un indirizzo email

Verifica la validità dell'indirizzo email.

Il metodo deve restituire true se l'email è valida, ovvero se soddisfa i criteri di formato corretti, altrimenti deve restituire false.

Returns
true se l'indirizzo email è valido, false altrimenti.
