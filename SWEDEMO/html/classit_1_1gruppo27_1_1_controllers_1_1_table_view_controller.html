void it.gruppo27.Controllers.TableViewController.deleteAll ( )

Eliminazione di tutti i contatti dalla rubrica.

Mostra un messaggio di conferma per chiedere all'utente se vuole procedere con l'eliminazione dell'intera rubrica.

Precondition
ci deve essere almeno un contatto salvato all'interno della rubrica
Postcondition
mostra una tableView vuota.

◆ initializeTable()

void it.gruppo27.Controllers.TableViewController.initializeTable ( )

Inizializza la tabella e imposta i dati.

Configura le colonne della tabella e assegna la lista osservabile dei contatti. Imposta anche un messaggio di placeholder se la tabella è vuota.

Precondition
tableView , nomeClm , surnameClm non possono essere null.
Postcondition
mostra una tableView vuota, con due colonne ,che cambia in base alle modifiche della lista osservabile dei contatti

◆ updateView()

void it.gruppo27.Controllers.TableViewController.updateView ( )

Aggiorna la vista della tabella.

Reimposta la lista osservabile legata alla tabella con i contatti correnti presenti nella rubrica.

Postcondition
all'utente viene mostrata una table view aggiornata in base alle operazioni eseguite.
