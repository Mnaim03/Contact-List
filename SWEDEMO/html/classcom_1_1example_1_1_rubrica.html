◆ leggiVCF()

Rubrica com.example.Rubrica.leggiVCF ( String filename) throws IOException, InvalidEmailException, InvalidNumberException

Legge un file .vcf (vCard) e carica i contatti al suo interno nella rubrica.

Questo metodo esamina un file .vcf che contiene informazioni sui contatti e le carica all'interno della rubrica. Ogni contatto viene analizzato per estrarre i dati come nome, cognome, numeri di telefono ed email. In caso di errori nel file, vengono lanciate eccezioni specifiche per segnalarli.

Parameters
filenameIl percorso del file .vcf da leggere. Il file deve essere nel formato vCard standard.
Returns
Una nuova rubrica che contiene i contatti letti dal file VCF.
Exceptions
IOExceptionSe si verifica un errore durante la lettura del file, come un file inesistente o accessibile.
InvalidEmailExceptionSe una o più email nel file non sono valide.
InvalidNumberExceptionSe uno o più numeri di telefono nel file non sono validi.
Precondition
Il file specificato deve esistere e deve essere accessibile in lettura.
Il file deve essere formattato correttamente come file VCF.
Postcondition
Se il file è letto correttamente, i contatti vengono aggiunti alla rubrica.
Se ci sono errori nei dati (email o numeri non validi), viene sollevata l'eccezione corrispondente.
See also
InvalidEmailException Per informazioni sulle eccezioni relative alla validità delle email.
InvalidNumberException Per informazioni sulle eccezioni relative alla validità dei numeri di telefono.

◆ salvaVCF()

void com.example.Rubrica.salvaVCF ( String filename) throws IOException

Esporta i contatti della rubrica in un file .vcf (vCard).

Questo metodo consente di esportare tutti i contatti presenti nella rubrica in un file con formato VCF (vCard). Il formato VCF è uno standard utilizzato per memorizzare informazioni sui contatti, come nome, numero di telefono, email e altre informazioni pertinenti.

Parameters
filenameIl percorso del file in cui verranno salvati i contatti in formato .vcf. Il file verrà creato nella posizione specificata.
Precondition
La rubrica non è vuota
Postcondition
Viene creato un nuovo file .vcf contenente i dati della rubrica nella posizione specificata
Exceptions
IOExceptionSe si verifica un errore durante la scrittura del file, ad esempio se la destinazione non è accessibile o se si verifica un errore di I/O.
See also
Rubrica Per maggiori dettagli su come la rubrica gestisce i contatti.
