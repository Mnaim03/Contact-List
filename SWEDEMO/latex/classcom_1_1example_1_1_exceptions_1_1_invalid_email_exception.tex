\doxysubsection{Detailed Description}
La classe {\ttfamily \doxylink{classcom_1_1example_1_1_exceptions_1_1_invalid_email_exception}{Invalid\+Email\+Exception}} è un\textquotesingle{}eccezione controllata personalizzata che estende la classe {\ttfamily Exception}. Questa eccezione viene utilizzata per segnalare errori relativi a un\textquotesingle{}email non valida, come quando un\textquotesingle{}email non soddisfa i criteri di validità definiti dal sistema.

Un esempio di quando questa eccezione potrebbe essere sollevata include\+:
\begin{DoxyItemize}
\item L\textquotesingle{}aggiunta di un\textquotesingle{}email già presente nella lista delle email di un contatto.
\item L\textquotesingle{}inserimento di un\textquotesingle{}email che non rispetta il formato corretto (ad esempio, mancanza del carattere "{}@"{} o di un dominio valido). 
\end{DoxyItemize}

La classe estende la classe {\ttfamily Exception}, il che significa che è una {\itshape checked exception}, ovvero deve essere gestita esplicitamente nel codice tramite un blocco {\ttfamily try-\/catch} o essere dichiarata nel metodo con {\ttfamily throws}.
